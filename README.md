# ![](icons/ytMusicIcon.png) Youtube Music Control 

Firefox addon to control active youtube music tab from any other tab.   
Features include:   
⋅ Play/pause track   
⋅ Go to next/previous track   
⋅ Change volume   
⋅ Display track name when hovering over album art
